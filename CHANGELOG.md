# 更新日志

所有显著更改将被记录在此文件中。

## [1.0.0-beta.0] - 2025-12-17

### 新增

- 初始项目结构和核心功能
- Express.js 服务器基础架构
- 安全中间件（Helmet, CORS, 速率限制）
- 性能优化（压缩，缓存）
- 专业日志系统（Pino）
- 全局错误处理
- 健康检查端点
- 静态文件服务支持
- 环境变量配置系统
- 命令行参数支持
- 端口自动检测和冲突处理
- 优雅关机处理

### 功能

- 完全 ES Modules 支持
- 生产环境就绪配置
- 可定制中间件和路由
- 多环境支持（开发、生产、测试）
- 结构化日志和请求日志
- 自动缓存策略
- 本地 IP 地址检测

### 修复

- 修复端口冲突检测问题
- 修复 Windows 路径解析问题
- 修复日志格式化问题
- 修复中间件加载顺序问题

### 变更

- 升级到 Express 5.x
- 升级到 Node.js 20+ 兼容性
- 优化默认配置值
- 改进错误消息和日志格式
